# Plan 03-02: Navigation + KPI Cards

**Phase:** 03 — Dashboard & Analytics
**Wave:** 2 of 4
**Goal:** Add app navigation (Entry <-> Dashboard) and build the KPI card row — the "morning glance" that the owner sees first.

## Tasks

| # | Task | Key Files |
|---|------|-----------|
| 1 | Create bottom navigation bar component | `client/src/components/BottomNav.tsx` |
| 2 | Add `/dashboard` route and update App.tsx with nav | `client/src/App.tsx` |
| 3 | Create useDashboard hook (fetches summary + manages area filter) | `client/src/features/dashboard/hooks/useDashboard.ts` |
| 4 | Create KpiCard component | `client/src/features/dashboard/components/KpiCard.tsx` |
| 5 | Create BusinessAreaFilter component (pill toggle) | `client/src/features/dashboard/components/BusinessAreaFilter.tsx` |
| 6 | Create DashboardPage with KPI row + filter | `client/src/features/dashboard/DashboardPage.tsx` |

## Task Details

### Task 1: BottomNav
Fixed bottom bar with two items: "Entry" (icon: PenLine) and "Dashboard" (icon: BarChart3). Highlights active route. Dark themed: bg `#252220`, border-t `#3D3530`, active icon `#E7C76A`, inactive `#7A6F63`.

Positioned fixed at bottom, 56px height. Content area needs bottom padding to avoid overlap.

### Task 2: Routing Update
- Add `<Route path="/dashboard" element={<DashboardPage />} />`
- Wrap Routes in a layout that includes BottomNav
- Entry page stays at `/` (default)

### Task 3: useDashboard Hook
```typescript
interface DashboardState {
  area: 'all' | 'retail' | 'playroom_cafe'
  month: string // YYYY-MM
}
```
- Fetches `/api/dashboard/summary?month=X&area=Y` via TanStack Query
- Exposes `summary`, `isLoading`, `area`, `setArea`, `month`
- Default month = current month, area = "all"

### Task 4: KpiCard Component
Reusable card showing:
- Label (e.g., "Today's Revenue")
- Value (e.g., "245.50") with euro sign
- Delta badge (e.g., "+12.3%") — green for positive, red for negative
- Optional suffix badge (e.g., "Estimated" for profit basis)

Dark themed card: bg `#252220`, border `#3D3530`, value text `#E8E0D4`, label `#9A8E82`, delta green `#6ECF8E`, delta red `#D4564E`.

### Task 5: BusinessAreaFilter
Horizontal pill row: "All" | "Retail" | "Playroom & Cafe"
- Active pill: bg `#3D3530`, text `#E8E0D4`
- Inactive: text `#7A6F63`, hover `#C4B8A8`
- Rounded-full pills with transition
- "Parties" pill shown but disabled/dimmed (Phase 4)

### Task 6: DashboardPage Assembly
```
[BusinessAreaFilter]
[KpiCard: Today] [KpiCard: MTD Revenue] [KpiCard: Gross Profit]
[... charts come in waves 3-4 ...]
```
- Title: "Dashboard" in Cormorant Garamond
- Subtitle showing current month name (e.g., "March 2026")
- KPI cards in a responsive row (flex-wrap on narrow screens)
- `max-w-[600px] mx-auto` (slightly wider than entry page to fit charts)
