# Plan 03-03: Revenue Trend Chart + Category Breakdown

**Phase:** 03 — Dashboard & Analytics
**Wave:** 3 of 4
**Goal:** Build the two primary charts — monthly revenue trend (hero chart) and category income breakdown. These are the visual core of the dashboard.

## Tasks

| # | Task | Key Files |
|---|------|-----------|
| 1 | Create useChartData hook (fetches trend + categories) | `client/src/features/dashboard/hooks/useChartData.ts` |
| 2 | Create RevenueTrendChart component (area chart) | `client/src/features/dashboard/components/RevenueTrendChart.tsx` |
| 3 | Create CategoryBreakdownChart component (horizontal bars) | `client/src/features/dashboard/components/CategoryBreakdownChart.tsx` |
| 4 | Integrate charts into DashboardPage | `client/src/features/dashboard/DashboardPage.tsx` |

## Task Details

### Task 1: useChartData Hook
- Fetches `/api/dashboard/trend?month=X&area=Y` — daily revenue series (current + previous month)
- Fetches `/api/dashboard/categories?month=X&area=Y` — per-category totals
- Both queries keyed on `[month, area]` for TanStack Query cache
- Returns `{ trend, categories, isLoading }`

### Task 2: RevenueTrendChart (Hero Chart)
Recharts `AreaChart` or `ComposedChart`:
- **Current month line**: Honey Yellow `#E7C76A` with gradient fill (fade to transparent)
- **Previous month line**: Muted `#7A6F63` dashed, no fill (subtle comparison)
- X-axis: day of month (1, 2, ... 31), tick color `#7A6F63`
- Y-axis: euro values, tick color `#7A6F63`
- Grid lines: very subtle `#3D3530`
- Tooltip: dark bg `#2A2622`, border `#3D3530`, showing date + value
- Responsive: width 100%, height ~240px
- Legend: "This month" / "Last month" — small, below chart

Chart card wrapper: bg `#252220`, rounded-xl, border `#3D3530`, padding, title "Revenue Trend" in `#C4B8A8`.

Empty state: if no data, show "No entries yet for [month]" with muted text.

### Task 3: CategoryBreakdownChart
Recharts `BarChart` (horizontal):
- Bars colored by business area:
  - Retail categories: Honey Yellow variants (`#E7C76A`, `#D4B55A`, `#C1A34A`)
  - Playroom categories: Dusty Mint variants (`#BFD8D2`, `#A8C5BE`, `#91B2AA`)
- Category names on Y-axis (left), revenue values on X-axis
- Bar height ~28px, gap 4px
- Tooltip showing category name + revenue + profit
- Responsive, height = categories.length * 44px (min 200px)

Card wrapper matching RevenueTrendChart style.

### Task 4: Dashboard Integration
Add below KPI cards:
```
[RevenueTrendChart — full width]
[CategoryBreakdownChart — full width]
```
Both respond to the active area filter from useDashboard.
