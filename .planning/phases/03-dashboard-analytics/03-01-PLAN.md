## Task Details

### Task 1: Summary Endpoint
`GET /api/dashboard/summary?month=YYYY-MM&area=all`

Returns:
```json
{
  "todayRevenue": 245.50,
  "mtdRevenue": 3420.00,
  "mtdProfit": 1580.00,
  "profitBasis": "estimated",
  "prevMonthRevenue": 3100.00,
  "prevMonthProfit": 1400.00,
  "revenueDelta": 10.3,
  "profitDelta": 12.9
}
```

SQL logic:
- `todayRevenue`: SUM(revenue) WHERE entry_date = today, filtered by area if not "all"
- `mtdRevenue`: SUM(revenue) WHERE entry_date BETWEEN month-start AND today
- `mtdProfit`: SUM(revenue) - SUM(effective_cost) where effective_cost = COALESCE(cost_amount, revenue * cost_margin_pct)
- `profitBasis`: "exact" if ALL entries in range have non-null cost_amount, otherwise "estimated"
- `prevMonth*`: same queries for previous calendar month (full month, not MTD)
- `revenueDelta`: ((mtdRevenue - prevMonthRevenue) / prevMonthRevenue) * 100
- `area` filter: JOIN categories WHERE business_area = param (or no filter for "all")

### Task 2: Daily Trend Endpoint
`GET /api/dashboard/trend?month=YYYY-MM&area=all`

Returns:
```json
{
  "currentMonth": [
    { "day": 1, "date": "2026-03-01", "revenue": 180.50 },
    { "day": 2, "date": "2026-03-02", "revenue": 220.00 }
  ],
  "previousMonth": [
    { "day": 1, "date": "2026-02-01", "revenue": 165.00 }
  ]
}
```

SQL: GROUP BY entry_date, SUM(revenue), filtered by month and optionally by business_area via JOIN.

### Task 3: Category Breakdown Endpoint
`GET /api/dashboard/categories?month=YYYY-MM&area=all`

Returns:
```json
{
  "categories": [
    {
      "categoryId": 1,
      "categoryName": "Toys",
      "displayName": "Toys",
      "businessArea": "retail",
      "revenue": 890.00,
      "cost": 489.50,
      "costBasis": "estimated",
      "profit": 400.50
    }
  ]
}
```

SQL: GROUP BY category_id, SUM(revenue), SUM(effective_cost), JOIN categories for names and business_area.

### Task 4: Kids Trend Endpoint
`GET /api/dashboard/kids?days=30`

Returns:
```json
{
  "entries": [
    { "date": "2026-03-01", "count": 12 },
    { "date": "2026-03-02", "count": 8 }
  ]
}
```

SQL: SELECT from kids_entries WHERE entry_date >= (today - N days), ORDER BY entry_date.

### Task 5: Mount Route
Add `import dashboardRoute from './routes/dashboard'` and `app.route('/api/dashboard', dashboardRoute)` to server index.
